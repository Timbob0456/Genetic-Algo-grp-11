# Genetic-Algo-grp-11

Genetic algorithm implementation for optimizing based on the Traveling Salesman Problem

## Project Overview
This project uses a genetic algorithm to find the best route going through each city only once and back to the starting point. The algorithm uses selection, crossover, mutation, and elitism to change a group of routes over many generations.

## Project File

- `genetic_algorithm.py` — main Python file

## Requirements

- Python 3.x
- NumPy

## Install Dependencies

Open a terminal in the project folder and run:

pip install numpy

## How to Run

1. Download or clone the project folder.
2. Open a terminal in the folder containing `genetic_algorithm.py`.
3. Run the program with:

python genetic_algorithm.py

## Features
- Random permutations for population initialization
- Permutation represents route encoding
- Total route distance determines fitness evaluation
- Parent selection based on tournament selection
- Offspring generated by one-point crossover
- Mutation swapped to maintain diversity
- Best solution preserved through elitism
- Evolution process across multiple generations
- Best route per generation tracked and printed
- Visualization of the final best route through text/graph output

## Expected Output

The program will print the distances and their total distances, and various randomly generated routes.
